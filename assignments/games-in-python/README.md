
# 📘 Assignment: Games in Python

## 🎯 Objective

Build a complete Hangman game in Python to practice strings, conditionals, loops, and user input handling.

## 📝 Tasks

### 🛠️ Hangman Game Core Logic

#### Description

Implement the main Hangman game loop that selects a random word, accepts player guesses, and tracks game progress.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Display the current word progress as `_ _ _` with correctly guessed letters revealed
- Track and display incorrect guesses remaining
- Prevent repeating already guessed letters and inform the player
- End with a win when the word is fully guessed, or a loss when attempts are exhausted

### 🛠️ Player Interaction and Messaging

#### Description

Add user-friendly input/output, including prompts, progress updates, and result messages.

#### Requirements
Completed program should:

- Prompt the player for a letter guess and validate one-letter input
- Show the letters guessed so far (correct and incorrect)
- Display clear win/lose messages at game end
- Optionally offer to restart the game for another round
